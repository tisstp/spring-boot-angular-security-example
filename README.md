# spring-boot-angular-security-example
Security Demo with Spring boot and Angular


## Run Application
```bash
# Angular
$ cd angular 
$ yarn install
$ yarn run start


# spring boot
$ gradle bootRun
$ ./gradlew tasks --stacktrace
```


## CLI
```bash
# spring cli
$ sdk install springboot
$ spring --version
$ spring help run

# angular cli


# gradle cli
```
